visualization.topoplot module¶

Created on Jun 12, 2018

Creates a topoplot from provided data, indicating either size of change or size of change and significance.

class visualization.topoplot.topoplot(mode)¶

topoplot generation class. Initialition costs a couple of seconds due to mask generation. Performance advice: if possible, only generate a single topoplot object.

run(values, ch_name_list, omit_channels=None, substitute_channels=None, v_min=None, v_max=None, v_border_values=[], v_border_labels=[''], file_path=None, screen_channels=False, annotate_ch_names=False)¶

Plots a 2D topomap

Parameters

  • values – May either be a N x 3 or N x 1 matrix. Dimensions #2 (boolean only) may be used to indicate significance before multiple comparison correction and dimensions #3 (boolean only) may be used to indicate significance after multiple comparison correction.

  • ch_names – Names of the individual channels. Used for the spatial positioning of channels and the annotation of channels.

  • omit_channels – A list which channels are to be omitted. Channels are identified via names matching the ones specified in ch_names.

  • substitute_channels – A list of dictionaries. Each dictionary contains a ‘tgt’ section with a single string defining the channel to be substituted and a second ‘src’ section which contains a list of strings, defining channel names which are used to substitute the ‘tgt’ channel.

  • v_min – Minimal value on the color bar. If None, v_min is chosen as the minimum value within the data.

  • v_max – Maximuim value on the color bar. If None, v_max is chosen as the maximum value within the data.

  • v_border_values – Where to put new ticks onto the color bar. v_min and v_max are always added as values. The number of labels defined in v_border_labels must be exactly one element larger than the number of elements in v_border_values.

  • v_border_labels – Labels for the ticks on the color bar. The number of labels defined in v_border_labels must be exactly one element larger than the number of elements in v_border_values.

  • file_path – Path (including file name and file ending) were the file is stored. In case of None, the file is not saved.

  • screen_channels – If true, channels are not drawn as a smoothed 2D plane, but a voroni diagram easening the identification of individual unexpected results.

  • annotate_ch_names – If true, channels get annotate with their individual names.

Returns

The figure and the axes object to easen the inclusion of a plot into a larger picture.
